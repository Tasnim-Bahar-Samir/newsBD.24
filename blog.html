The scope of a var variable is functional scope. The scope of a let variable is block scope. The scope of a const variable is block scope.Var and let variable can be updated. But The user cannot update the const variable once it is declared.

Inside of a regular JavaScript function, this value is dynamic.But The behavior of this inside of an arrow function differs considerably from the regular function's this behavior. The arrow function doesn't define its own execution context. Inside the body of a regular function, arguments is a special array-like object containing the list of arguments with which the function has been invoked. On the other side, no arguments special keyword is defined inside an arrow function.
